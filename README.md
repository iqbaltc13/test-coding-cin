# Test Coding CIN



## Project setup
```
1. Clone This project
2. cd test-coding-cin (or directory of cloned this project)
3. create a database on mysql
3. cp .env.example .env
4. set the environment variables (config database, app kee , etc) on .env file
5. npm install
6. import api collection (Test Coding CIN.postman_collection.json) to postman

```

### Run
```
1. node server.js
2. run api collection on postman

```
